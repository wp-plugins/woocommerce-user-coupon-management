Attach coupons to user(s)

== Description ==

This plugin allows you to manage the WooCommerce coupons per users.

== Installation ==

1. Upload `woocommerce-coupon-user-management` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Coupons section in WooCommerce and attach user(s) to coupons
